# GitHub 로그인 방법

GitBackup을 사용하려면 먼저 GitHub 계정을 연결해야 합니다.

로그인하지 않으면 폴더 선택, 가져오기, 동기화 기능을 사용할 수 없습니다.

## 로그인 순서

1. `GitHub 로그인`을 누릅니다.
2. 브라우저가 열리면 GitHub 로그인 페이지로 이동합니다.
3. 앱에 표시된 코드를 브라우저에 입력합니다.
4. 연결 권한을 승인합니다.
5. 앱으로 돌아옵니다.

완료되면 로그인 상태에 계정 이름이 표시됩니다.

## 코드 입력 화면이 보일 때

앱 화면에 큰 글자로 코드가 보일 수 있습니다.

이 코드는 일회용 코드입니다. 그대로 브라우저에 입력하면 됩니다.

코드 만료 시간이 지나면 `새 코드 받기`를 눌러 다시 진행하세요.

## 로그인 후 확인할 것

- `GitHub 연결` 카드에 계정 이름이 보이는지 확인합니다.
- 대시보드 상단에서 `GitHub 로그인` 대신 백업 관련 버튼이 보이는지 확인합니다.
- `설정` 화면에서 `GitHub 로그인` 대신 `로그아웃` 버튼이 보이는지 확인합니다.

## 로그아웃 방법

아래 두 곳 중 한 곳에서 로그아웃할 수 있습니다.

- 대시보드의 `로그아웃`
- `설정` 화면의 `로그아웃`

## 로그인되지 않을 때

- 인터넷 연결을 확인하세요.
- 브라우저에서 GitHub 로그인과 승인까지 끝냈는지 확인하세요.
- 코드가 만료되었으면 새 코드를 받으세요.
- 앱을 완전히 닫은 뒤 다시 열어 보세요.
- 문제가 계속되면 [문제 해결](/docs/troubleshooting.md)을 확인하세요.
