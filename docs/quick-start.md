# 빠른 시작 가이드

처음 사용하는 경우 아래 순서대로 진행하세요.

## 1. 앱 설치와 실행

Windows 설치 파일 `.exe`를 실행해 GitBackup을 설치합니다.

설치가 끝나면 앱을 실행합니다.

## 2. GitHub 로그인

`GitHub 로그인`을 누릅니다.

브라우저가 열리면 안내에 따라 로그인하고 연결을 승인합니다.

앱 화면에 코드가 보이면 브라우저에서 그 코드를 입력합니다.

## 3. 백업 방식 선택

원하는 작업을 선택합니다.

- 새로 백업을 시작하려면 `새 백업 감시`
- 이미 GitHub에 있는 백업을 가져오려면 `기존 백업 가져오기`

## 4. 폴더 또는 저장 위치 선택

새 백업이면 백업할 폴더를 고릅니다.

기존 백업이면 저장소를 가져올 위치를 고릅니다.

## 5. 저장소 연결

새 백업이면 아래 둘 중 하나를 선택합니다.

- `새 비공개 저장소 만들기`
- `기존 저장소 가져오기`

기존 백업이면 저장소를 선택하거나 주소를 입력해 가져오기를 진행합니다.

## 6. 감시 시작

`감시 시작`을 누르면 GitBackup이 폴더 변경을 확인합니다.

파일을 바꾸면 잠시 뒤 자동으로 동기화됩니다.

## 빠른 확인

아래 항목이 보이면 정상입니다.

- 로그인 상태: 연결됨
- 연결 저장소: 표시됨
- 감시 서비스: `감시 중`, `동기화 중`, `최근 동기화 완료` 중 하나로 표시됨

## 잘 안될 때

- 로그인되지 않으면 [GitHub 로그인 방법](/docs/github-login.md)을 확인하세요.
- 가져오기에 실패하면 [기존 저장소 가져오기](/docs/import-existing-repository.md)를 확인하세요.
- 오류가 계속되면 [문제 해결](/docs/troubleshooting.md)을 확인하세요.
