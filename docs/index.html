GitHub 계정이 필요한 이유도 간단합니다

GitBackup은 내 파일을 GitHub 저장소에 보관하는 방식입니다. 그래서 계정 연결이 필요하지만, 그만큼 기존 백업을 다시 가져오거나 저장소를 명확하게 관리하기 쉬워집니다.

  • 내 저장소에 직접 연결
  • 기존 백업 다시 가져오기 가능
  • 백업 대상을 명확하게 관리
FAQ

자주 묻는 질문

Git을 잘 몰라도 사용할 수 있나요?

네. Git 명령을 직접 다루는 대신, 로그인과 버튼 중심 흐름으로 백업을 시작할 수 있게 구성했습니다.

누구를 위한 앱인가요?

중요한 폴더를 쉽고 빠르게 백업하고 싶은 일반 Windows 사용자를 위한 앱입니다.

GitHub 계정이 꼭 필요한가요?

네. GitBackup은 GitHub 저장소에 파일을 보관하는 방식이므로 GitHub 계정 연결이 필요합니다.

기존에 GitHub에 올려둔 백업도 다시 가져올 수 있나요?

가능합니다. 가져오기 화면에서 저장소를 불러오거나 주소를 입력해 연결할 수 있습니다.

Windows에서만 사용할 수 있나요?

이 페이지와 현재 다운로드 안내는 Windows 사용자를 기준으로 구성되어 있습니다.
