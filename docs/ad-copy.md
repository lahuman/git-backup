## 4. 커뮤니티 홍보 글

### 짧은 버전
GitBackup은 일반 Windows 사용자를 위한 쉬운 파일 백업 앱입니다.  
GitHub에 로그인하고 폴더만 선택하면, 중요한 자료를 쉽고 빠르게 백업할 수 있습니다.

### 중간 버전
중요한 폴더를 수동으로 복사해두는 방식이 번거로워서 GitBackup을 만들었습니다.  
GitBackup은 일반 Windows 사용자가 Git 명령 없이도 폴더를 GitHub에 백업할 수 있도록 만든 앱입니다.  
로그인하고, 폴더를 고르고, 감시를 시작하면 더 간단하게 백업 흐름을 유지할 수 있습니다.

### 출시 소개 버전
Windows에서 중요한 폴더를 쉽고 빠르게 백업하고 싶다면 GitBackup을 써보세요.  
복잡한 Git 도구 대신, 버튼 중심 UI로 GitHub 로그인, 저장소 연결, 폴더 감시, 기존 백업 가져오기를 처리할 수 있습니다.  
개발자가 아니어도 이해하기 쉬운 흐름으로 만든 백업 앱입니다.
