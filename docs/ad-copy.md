# GitBackup 광고 문구

## 한 줄 소개

- Windows 사용자를 위한 쉽고 빠른 파일 백업 앱
- 복잡한 설정 없이 폴더만 고르면 시작되는 백업
- 누구나 바로 시작할 수 있는 GitHub 기반 파일 백업

## 랜딩 페이지 상단 문구

### 버전 1
Windows에서 중요한 폴더를 쉽고 빠르게 백업하세요.

### 버전 2
복잡한 백업 대신, 로그인하고 폴더만 선택하세요.

### 버전 3
누구나 바로 시작할 수 있는 쉬운 파일 백업 앱.

## X용 짧은 문구

## 한국어 SNS 문구

### 버전 1
중요한 파일, 이제 더 쉽게 백업하세요. GitBackup은 Windows 사용자를 위한 쉽고 빠른 파일 백업 앱입니다.

### 버전 2
복잡한 설정 없이 로그인하고 폴더만 선택하세요. GitBackup으로 누구나 파일 백업을 쉽게 시작할 수 있습니다.

### 버전 3
수동 복사보다 더 편하게. GitBackup으로 중요한 폴더를 쉽고 빠르게 백업하세요.
