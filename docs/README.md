# GitBackup GitHub Pages 패키지

이 폴더는 GitHub Pages에 바로 올릴 수 있는 정적 랜딩 페이지 파일입니다.

## 포함 파일
- `index.html`
- `styles.css`
- `assets/` 이미지들
- `ad-copy.md`

## 배포 방법
1. 저장소에 `docs` 폴더가 있다면 이 파일들을 `docs/`에 넣습니다.
2. GitHub 저장소 설정에서 Pages 게시 소스를 `main` 브랜치의 `/docs`로 지정합니다.
3. 배포 후 랜딩 페이지 URL을 광고 링크로 사용합니다.

## 수정 권장 항목
- 다운로드 버튼 링크를 실제 설치 파일 또는 Releases 링크로 조정
- 페이지 하단 문구와 GitHub 저장소 링크 확인
- 필요하면 FAQ와 소개 문구를 더 짧게 다듬기
