# GitBackup Pages Redesign

GitHub Pages용 랜딩 페이지 수정본입니다.

## 포함 파일

- `index.html`
- `styles.css`
- `assets/`
- `ad-copy.md`

## 배포 방법

1. 이 폴더의 파일을 저장소 `docs/` 아래로 복사합니다.
2. GitHub 저장소에서 `Settings > Pages` 로 이동합니다.
3. `Deploy from a branch` 를 선택합니다.
4. 브랜치를 `main`, 폴더를 `/docs` 로 선택합니다.
5. 저장 후 배포 주소를 확인합니다.

## 권장 링크

다운로드 버튼은 아래 주소를 사용합니다.
