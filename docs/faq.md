# 자주 묻는 질문

## GitBackup은 무엇인가요

내 폴더를 GitHub에 백업할 수 있게 도와주는 Windows용 앱입니다.

## Git을 알아야 하나요

아니요. 기본 사용에는 복잡한 Git 지식이 필요하지 않습니다.

## 별도로 Git을 설치해야 하나요

보통은 필요하지 않습니다.

GitBackup은 별도 Git 설치 없이 사용할 수 있도록 준비되어 있습니다.

## GitHub 계정이 꼭 필요한가요

네. GitBackup은 GitHub에 백업을 저장하므로 GitHub 계정이 필요합니다.

## 백업은 자동으로 되나요

네. 폴더 감시를 시작하면 변경 내용을 자동으로 확인하고 동기화합니다.

## 바로 백업하고 싶으면 어떻게 하나요

`지금 동기화`를 누르면 됩니다.

## 새 저장소도 만들 수 있나요

네. 폴더를 고른 뒤 저장소 이름을 입력하고 `새 비공개 저장소 만들기`를 누르면 됩니다.

## 이미 GitHub에 백업이 있으면 새로 만들 필요가 있나요

아니요. `기존 백업 가져오기`로 그대로 가져오면 됩니다.

## 웹에서 만든 저장소도 사용할 수 있나요

네. 가능합니다.

가장 쉬운 방법은 `기존 백업 가져오기`로 빈 폴더에 가져오는 것입니다.

## 이미 GitHub와 연결된 폴더도 쓸 수 있나요

네. 대시보드에서 폴더를 선택하면 기존 GitHub 연결을 찾을 수 있습니다.

다만 HTTPS 연결이 더 안전합니다.

## SSH 저장소 주소도 되나요

현재는 HTTPS 연결을 권장합니다.

`git@github.com:` 형식보다 `https://github.com/...` 형식이 더 잘 맞습니다.

## 큰 파일도 백업되나요

아주 큰 파일은 경고가 표시되거나 업로드가 어려울 수 있습니다.

특히 100MB가 넘는 파일은 먼저 확인하는 것이 좋습니다.

## 여러 컴퓨터에서 같은 백업을 쓸 수 있나요

가능합니다.

한 컴퓨터에서 백업을 만들고, 다른 컴퓨터에서는 `기존 백업 가져오기`를 사용하세요.

## 로그인은 왜 브라우저에서 하나요

GitHub 계정 보안을 유지하면서 로그인 절차를 쉽게 진행하기 위해서입니다.
